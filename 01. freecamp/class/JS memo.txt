변수 : 어떤 관계나 범위 안에서 여러가지 값으로 변할 수 있는 수
상수 : 변하지 아니하는 일정한 값을 가진 수나 양

변수, 상수는 데이터를 담는 공간이다.
단순한 텍스트일수도, 몇백만의 코드일수도 있다.

변수를 만드는 방법에는 여러가지가 있지만 
 var, let, const 등이 있다.
 - var : 이름 중복(재선언) 가능 / 데이터 수정(재할당) 가능
 ㄴ 코드가 길어지면 상단에서의 내용과 중복되어 에러 가능성이 크기 때문에 실무에서는 거의 사용되지 않는다.
 - let : 이름 중복(재선언) 불가 / 데이터 수정(재할당) 가능
 ㄴ 변수를 위해 주로 사용된다.
 - const : 이름 중복(재선언) 불가 / 데이터 수정(재할당) 불가
 ㄴ 상수를 위해 주로 사용된다.

크롭 개발자 도구 설명
 - 요소(Elements) : HTML을 분석하고 수정해볼 수 있는 독수
 - 콘솔(Console) : 현재 로딩된 페이지에서 자바스크립트를 시험하거나 로그/오류 메세지 등을 확인할 수 있는 도구
 - 소스(Resources) : 현재 로딩된 페이지에서 사용된 토큰, 이미지, 동영상을 포함한 모든소스를  열람할 수 있는 도구
 - 네트워크(Network) : 서버와의 통신 내역을 보여주는 도구
 - 성능(Audits) : 웹 어플리케이션의 성능을 향상시킬 방법을 컨설팅해주는 도구 - 잘 사용 안함.

배열은 아래처럼 대괄호 안에 테이터를 넣고 쉼표로 각 데이터를 구분할 수 있다.
 ㄴ let students=["데이터1", "데이터2", "데이터3", "데이터4"]
 ㄴ 배열은 빈 배열, 숫자 배열, 문자열 배열 등 기능, 정보, 코드 등을 넣을 수 있다.

배열에는 index라는 개념을 이해해야 한다.
 ㄴ 배열에 있는 각 데이터의 위치
 ㄴ index를 이용해서 배열을 구분한다.
 ㄴ index : 0부터 카운트
 ㄴ length : 1부터 카운트

 배열은 아래와 같이 사용한다.
 - const array=["배열"] -
  기본
  길이 : array.length : 길이를 세 준다.
  배열의 값 꺼내기 : array[숫자(인덱스)] : 값을 조회한다.
  배열 맨 뒤에 값 추가 : array.push()
  배열 맨 마지막 값 삭제 : arry.pop()
  배열 요소 정렬 : array.sprt()
  배열 데이터 확인 : array.includes(값) : 이 배열에 어떤 데이터가 있는지, 일치하는지 확인하는 기능이다.
  심화 
  배열 2개 연결 : array.concat(array2) : 기존 배열에 array2에 추가로 입력된 배열을 연결한다.
  배열을 문자로 만들기 : array.join()
  배열 자르기 : array.slice()
  배열 분리(나누기) : array.split("문자") : "문자"를 기준으로 앞 뒤에 있는 문자를 각각 배열로 변환
  배열에서 원하는 요소 뽑기 : array.filter()
  배열의 모든 요소 변경 : array.map()

소괄호가 붙어 있는 것들은 일종의 기능이다. 위에 있는 코드들은 자주 사용한다.
 기능 : 데이터를 직접 접근해서 가공하거나 변환하는 것을 말한다.
